Rekindle App
============

Stay in touch with the people you care about.

CS147 Class Project

Steps to run locally:

* `>> mongod`
* `>> node initDB.js` (optional)
* `>> node app.js`

Steps to run on heroku:

* `>> heroku addons:add mongolab` (for new app)
* `>> heroku run node initDB.js` (optional)
